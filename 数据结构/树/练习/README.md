

## 推荐练习顺序

1. `基本操作_练习.cpp`
- 目标：熟悉二叉树节点与指针连接、插入中间节点、删除并恢复连接。

2. `层序遍历_练习.py`
- 目标：掌握队列驱动的 BFS 模式。

3. `递归DFS_练习.py`
- 目标：掌握三种递归遍历模板（前/中/后序）。

4. `迭代DFS_练习.py`
- 目标：掌握栈模拟递归，尤其是中序和后序的迭代写法。

5. `数组表示_练习.py`
- 目标：理解“数组索引关系”下的树遍历与统一 DFS 框架。

6. `二叉搜索树_查找+插入_练习.py`
- 目标：掌握 BST 的有序查找与无重复插入。

7. `二叉搜索树_删除_练习.cpp`
- 目标：重点吃透 BST 删除三种情况，尤其双子节点替换逻辑。

8. `AVL树_练习.py`
- 目标：综合练习高度维护、平衡因子、四种旋转与插入/删除回溯平衡。
