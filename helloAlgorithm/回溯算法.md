- backtracking algorithm
- 暴力搜索，把可行解记录下来
- 示例
	- 通常采用“深度优先搜索”来**遍历解空间**
	- Question1:
	  给定一棵二叉树，搜索并记录所有值为 7 的节点，请返回节点列表。
		- Code1:
		  ```python
		  def pre_order(root: TreeNode):
		      """前序遍历：例题一"""
		      if root is None:
		          return
		      if root.val == 7:
		          # 记录解
		          res.append(root)
		      pre_order(root.left)
		      pre_order(root.right)
		  ```
	-
	- **尝试（attempt）与回退（backtracking）**
	- Question2:
	  在二叉树中搜索所有值为 7 的节点，**请返回根节点到这些节点的路径**。
		- Code:
		  ```python
		  def pre_order(root: TreeNode):
		      """前序遍历：例题二"""
		      if root is None:
		          return
		      # 尝试
		      path.append(root)
		      if root.val == 7:
		          # 记录解
		          res.append(list(path))
		      pre_order(root.left)
		      pre_order(root.right)
		      # 回退，恢复到原来的状态
		      path.pop()
		  ```
	-
	- 约束（constraint）条件用于“**剪枝（pruning）**”
			- Question3:
	  在二叉树中搜索所有值为 7 的节点，请返回根节点到这些节点的路径，**并要求路径中不包含值为 3 的节点**。
		- Code:
		  ```python
		  def pre_order(root: TreeNode):
		      """前序遍历：例题三"""
		      # 剪枝
		      if root is None or root.val == 3:
		          return
		      # 尝试
		      path.append(root)
		      if root.val == 7:
		          # 记录解
		          res.append(list(path))
		      pre_order(root.left)
		      pre_order(root.right)
		      # 回退
		      path.pop()
		  ```

- 框架：其中state为当前状态，choices为所有可做的选择
	- ```python
	  def backtrack(state: State, choices: list[choice], res: list[state]):
	      """回溯算法框架"""
	      # 判断是否为解
	      if is_solution(state):
	          # 记录解
	          record_solution(state, res)
	          # 不再继续搜索
	          return
	      # 遍历所有选择
	      for choice in choices:
	          # 剪枝：判断选择是否合法
	          if is_valid(state, choice):
	              # 尝试：做出选择，更新状态
	              make_choice(state, choice)
	              backtrack(state, choices, res)
	              # 回退：撤销选择，恢复到之前的状态
	              undo_choice(state, choice)
	  ```
-
-
- 优点：可以找到所有解。合理剪枝时效率高。
- 缺点：大规模或复杂问题效率低。
-
- 优化效率：
	- 剪枝
	- 启发式搜索：引入策略或估计值，优先搜索可能有解的路径
