## 借助入堆操作实现
- 首先创建一个空堆，然后遍历列表，依次对每个元素执行“入堆操作”
- ## 通过遍历堆化实现
	- 1. 将列表所有元素原封不动地添加到堆中，此时堆的性质尚未得到满足。
	- 2. 倒序遍历堆（层序遍历的倒序），依次对每个非叶节点执行“从顶至底堆化”。
	-
	- **每当堆化一个节点后，以该节点为根节点的子树就形成一个合法的子堆**
	- ```cpp
	  /* 构造方法，根据输入列表建堆 */
	  MaxHeap(vector<int> nums) {
	      // 将列表元素原封不动添加进堆
	      maxHeap = nums;
	      // 堆化除叶节点以外的其他所有节点
	      for (int i = parent(size() - 1); i >= 0; i--) {
	          siftDown(i);
	      }
	  }
	  ```
