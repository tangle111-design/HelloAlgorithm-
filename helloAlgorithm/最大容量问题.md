# Question
	-
		- 输入一个数组 \( ht \)，其中的每个元素代表一个垂直隔板的高度。数组中的任意两个隔板，以及它们之间的空间可以组成一个容器。
		- 容器的容量等于高度和宽度的乘积（面积），其中高度由较短的隔板决定，宽度是两个隔板的数组索引之差。
		- 请在数组中选择两个隔板，使得组成的容器的容量最大，返回最大容量。
		-
- # 问题分析
  collapsed:: true
	- 本题的状态为两个隔板的索引，记为 $$[i,j]$$ 。
		- $$cap[i,j] = min(ht[i],ht[j]) * (j-i)$$
		- 穷举复杂度为$$O(n^2)$$
-
- # 贪心策略
  collapsed:: true
	- ## 分析
	  collapsed:: true
							-
	- ## 策略
		- 初始化两指针，使其分列容器两端，每轮向内收缩短板对应的指针，直至两指针相遇。
-
- # Code O(n)
	- ```python
	  def max_capacity(ht: list[int]) -> int:
	      """最大容量：贪心"""
	      # 初始化 i, j，使其分列数组两端
	      i, j = 0, len(ht) - 1
	      # 初始最大容量为 0
	      res = 0
	      # 循环贪心选择，直至两板相遇
	      while i < j:
	          # 更新最大容量
	          cap = min(ht[i], ht[j]) * (j - i)
	          res = max(res, cap)
	          # 向内移动短板
	          if ht[i] < ht[j]:
	              i += 1
	          else:
	              j -= 1
	      return res
	  ```
-
- # 正确性证明
- 这些被跳过的状态实际上就是将长板 $$j$$ 向内移动的所有状态, 被跳过的状态都不可能是最优解，**跳过它们不会导致错过最优解**。
