# 快速幂
- ### 原理：
- 1. 任何正整数n（此处为指数）都可以表示为二进制形式（二进制思想）
- 2. 分解n，使得计算次数减少（分治思想）
- ### 例子：3**13
- ```
  计算 3^13：
  n = 13 (二进制: 1101)
  - 步骤分析：
  1. n=13(奇数): result = 1×3 = 3, a = 3² = 9, n=6
  2. n=6(偶数): result = 3, a = 9² = 81, n=3  
  3. n=3(奇数): result = 3×81 = 243, a = 81² = 6561, n=1
  4. n=1(奇数): result = 243×6561 = 1594323, a = 6561², n=0
  - 验证：3^13 = 1594323
  ```
- ```
  二进制视角：
  13 = 8 + 4 + 1 = 2^3 + 2^2 + 2^0
  3^13 = 3^8 × 3^4 × 3^1
  ```
- ### 代码实现：
- ```python
  def naive_power(a, n):
    '''朴素方法'''
      result = 1
      for i in range(n):
          result *= a
      return result
  # 时间复杂度: O(n)
  ```
- ```python
  def fast_power_iterative(a, n):
    '''快速幂 迭代'''
      result = 1
      while n > 0:
          if n & 1:  # n % 2 == 1
              result *= a
          a *= a     # 平方
          n >>= 1    # n //= 2
      return result
  # 时间复杂度: O(log n)
  ```
- ```python
  def fast_power_recursive(a, n):
    '''快速幂 递归'''
      if n == 0:
          return 1
      if n == 1:
          return a
      
      half = fast_power_recursive(a, n // 2)
      
      if n % 2 == 0:
          return half * half
      else:
          return half * half * a
  # 时间复杂度: O(log n)
  ```
- # 矩阵快速幂
- ```python
  def matrix_mult(A, B):
      """
      自定义矩阵乘法
      A, B: 二维列表表示的矩阵
      返回: A × B
      """
      n = len(A)
      m = len(B[0])
      p = len(B)
      
      # 初始化结果矩阵
      result = [[0] * m for _ in range(n)]
      
      # 矩阵乘法
      for i in range(n):
          for j in range(m):
              for k in range(p):
                  result[i][j] += A[i][k] * B[k][j]
      
      return result
  
  def matrix_power(matrix, power):
      """
      矩阵快速幂
      matrix: 二维列表表示的方阵
      power: 幂次
      返回: matrix^power
      """
      n = len(matrix)
      
      # 初始化结果矩阵为单位矩阵
      result = [[1 if i == j else 0 for j in range(n)] for i in range(n)]
      base = [row[:] for row in matrix]  # 复制矩阵
      
      # 快速幂算法
      while power > 0:
          if power & 1:  # 如果当前位为1
              result = matrix_mult(result, base)
          base = matrix_mult(base, base)  # 矩阵平方
          power >>= 1  # 右移一位
      
      return result
  ```
