# 哈希表练习顺序

下面是推荐练习路线（由浅入深）：

1. 哈希表常用操作_练习.py
2. 简单的哈希算法_练习.py
3. 简单哈希表实现_练习.py
4. 链式地址哈希表_练习.py
5. 开放寻址_练习.py
6. 链式地址哈希表_练习.c

## 每阶段目标

1. 常用操作：熟悉字典增删改查与遍历方式。
2. 哈希函数：理解不同哈希策略的构造方式。
3. 数组哈希表：理解“哈希函数 + 桶数组”的最小实现。
4. 链式地址法：掌握冲突处理、负载因子和扩容重哈希。
5. 开放寻址法：掌握线性探测、墓碑删除、查找与插入细节。
6. C 语言版本：强化指针、内存管理、链表操作与重哈希流程。

## 建议练习方式

1. 每实现一个函数就写一个最小测试用例（正常输入 + 边界输入）。
2. 先保证功能正确，再关注复杂度与代码风格。
3. 链式地址法和开放寻址法都完成后，对比两者优缺点并总结。
