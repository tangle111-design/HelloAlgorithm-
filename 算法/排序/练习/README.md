# 排序练习说明

这个目录是与上级“排序”源码一一对应的练习壳版本。
目标是按由浅入深、由易到难的顺序完成实现，再对照原实现复盘。

## 推荐练习顺序（由易到难）

### 第一阶段：建立排序直觉（最基础）

1. 冒泡排序_练习.py
2. 选择排序_练习.py
3. 插入排序_练习.py

说明：这一阶段重点是掌握“区间有序/无序”的循环不变量，以及交换、移动元素的基本手感。

### 第二阶段：分治思想入门（中等）

4. 归并排序_练习.py

说明：开始训练递归拆分 + 合并回写的完整流程，为后面的快速排序打基础。

### 第三阶段：快速排序专项（中高难）

5. 快速排序_练习.py
6. 快速排序_分区方案_练习.py
7. 快速排序_基准数优化_练习.py
8. 快速排序_递归深度优化_练习.py

说明：按“先会写 -> 会变体 -> 会优化”的顺序推进。

### 第四阶段：堆结构与原地排序（中高难）

9. 堆排序_练习.py

说明：重点理解完全二叉树下标映射、下沉操作和建堆过程。

### 第五阶段：非比较排序（进阶）

10. 计数排序_练习.py
11. 桶排序_练习.py
12. 基数排序_练习.py

说明：这三类排序对数据范围有假设，重在理解“用空间换时间”和“稳定性”的作用。

## 每轮练习建议

1. 先只看函数签名和步骤提示注释，独立完成实现。
2. 每题至少准备三组测试：随机、逆序、重复值。
3. 写完后对照上级目录原实现，记录差异点。
4. 第二遍练习时，重点优化边界条件与变量设计。

## 快速排序专项建议

1. 手动画一次 partition 的 i/j 指针移动轨迹。
2. 对同一组输入分别跑不同分区方案，比较返回分界点的含义。
3. 重点吃透三个问题：
   - 为什么递归边界是 left >= right。
   - 为什么 Hoare 分区后递归区间是 [low, j] 和 [j + 1, high]。
   - 为什么“只递归较短区间”能降低最大递归深度。
4. 对重复值很多的输入，优先练三路快排并观察效果。
