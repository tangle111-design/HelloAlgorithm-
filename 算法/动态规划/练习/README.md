# 动态规划练习说明

这个目录与上级“动态规划”源码一一对应，采用“步骤提示 + 独立实现”的练习方式。
建议按从入门到进阶的顺序完成。

## 推荐练习顺序

1. 爬楼梯问题_练习.py
2. 最小路径和问题_练习.py
3. 0-1背包问题_练习.py
4. 完全背包问题_练习.py
5. 编辑距离问题_练习.py
6. DP问题特性_练习.py

## 每轮练习建议

1. 先只看函数签名和步骤提示，独立完成推导与实现。
2. 每题至少准备三组测试：普通输入、边界输入、极端输入。
3. 实现后对照上级目录源码，重点复盘状态定义和转移方程。
4. 第二遍练习时，优先尝试空间优化版本。

## 动态规划专项复盘清单

1. 状态定义是否覆盖了“进度 + 约束”。
2. 状态转移是否只依赖已知子问题。
3. 边界条件是否完整（首行、首列、空输入等）。
4. 遍历顺序是否与依赖方向一致。
5. 能否做空间压缩，压缩后是否破坏依赖关系。
