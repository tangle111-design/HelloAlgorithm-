# 分治练习顺序建议

## 推荐顺序

1. 分治搜索_练习.py
2. 汉诺塔问题_练习.py
3. 构建二叉树问题_练习.py

## 为什么这样练

1. 分治搜索_练习.py
- 先熟悉最基础的分治递归模板：定义子问题、处理终止条件、合并答案（这里是直接返回）。
- 重点关注区间边界的变化（i、j、m）。

2. 汉诺塔问题_练习.py
- 训练“先拆分子问题再回到主问题”的递归过程。
- 重点理解 3 步递归搬运顺序，以及参数角色如何交换（src、buf、tar）。

3. 构建二叉树问题_练习.py
- 在分治基础上加入树结构构建，是综合度最高的一题。
- 重点掌握：
  - 如何通过 preorder 确定根节点。
  - 如何通过 inorder_map 切分左右子树区间。
  - 如何计算右子树根节点在 preorder 中的起点。

## 练习建议

- 第一轮：不看原题代码，按注释提示独立完成。
- 第二轮：把每个函数手动画递归调用树，检查参数变化是否正确。
- 第三轮：自己补 2~3 组测试用例，覆盖边界情况（空数组、单元素、极小规模）。
