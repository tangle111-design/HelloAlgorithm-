# 贪心练习说明

这个目录是与上级“贪心”源码一一对应的练习壳版本。
建议按“先理解贪心选择依据，再实现代码细节”的节奏完成。

## 推荐练习顺序（由易到难）

1. 最大容量问题_练习.py
2. 零钱兑换_练习.py
3. 分数背包_练习.py
4. 最大切分乘积问题_练习.py

## 顺序说明

- 最大容量问题：双指针写法直观，先建立“局部最优推进全局最优”的直觉。
- 零钱兑换：练习“每步取当前最优面值”的基本贪心流程与失败判定。
- 分数背包：在贪心中引入“单位价值排序 + 局部截断取值”。
- 最大切分乘积问题：需要处理余数分类与数学推导，思维抽象层次更高。

## 练习建议

1. 每题先手写步骤，再落代码，最后补测试。
2. 至少准备三类用例：
   - 基础用例（最小输入/常规输入）
   - 边界用例（空、极小、极大、临界值）
   - 反例用例（验证你写的贪心策略是否真的成立）
3. 完成后对照上级目录原实现，重点复盘：
   - 你的贪心选择依据是什么
   - 为什么这一步不会错过全局最优
   - 边界条件是否覆盖完整
