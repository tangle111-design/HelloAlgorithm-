# 算法学习顺序

这个目录建议按“先掌握基础技巧，再过渡到递归、剪枝和动态规划”的顺序练习。

## 推荐练习顺序

1. 排序
- 先从最基础的比较排序开始，建立对“交换、移动、分治”的直觉。
- 重点理解：冒泡、选择、插入、归并、快速排序，再到堆排序和非比较排序。

2. 搜索
- 练习二分查找及其边界写法，先把“有序区间上的判断”吃透。
- 重点理解：插入点、左边界、右边界、哈希优化策略。

3. 分治
- 在排序和二分的基础上，进入更一般的分而治之思想。
- 重点理解：递归拆分、合并结果、构建二叉树、汉诺塔。

4. 回溯
- 练习用递归搜索所有可能，并通过剪枝控制搜索空间。
- 重点理解：全排列、子集、组合搜索、N 皇后。

5. 贪心
- 训练“局部最优推出全局最优”的判断能力。
- 重点理解：分数背包、零钱兑换、最大切分乘积、最大容量。

6. 动态规划
- 放在最后系统练习，重点建立状态定义和转移方程的能力。
- 重点理解：爬楼梯、最小路径和、0-1 背包、完全背包、编辑距离。

## 建议

1. 每一类算法先手写一遍，再去对照源码。
2. 先把基础题做稳，再做同类题的练习文件。
3. 动态规划不要急着跳过前面部分，前面的排序、搜索、分治会直接帮助理解。
