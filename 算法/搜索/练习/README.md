# 搜索模块练习说明

本目录是“搜索”章节的练习壳文件，保留了函数接口与关键步骤提示，适合你手写补全。

## 推荐练习顺序

1. 二分查找_练习.cpp  
   目标：熟悉最基础的二分查找模板（双闭区间）。

2. 二分查找插入点_练习.cpp  
   目标：掌握“查找失败时返回插入位置”的思路，并区分“无重复/有重复”的处理。

3. 二分查找边界_练习.cpp  
   目标：学会通过“最左插入点”推导最左/最右边界，理解边界条件校验。

4. 哈希优化策略_练习.cpp  
   目标：掌握哈希表将两数之和优化到 O(n) 的写法，重点是“先查后写”。

## 练习建议

- 第 1 轮：先不看原始实现，按注释提示独立补全。
- 第 2 轮：对照原文件修正边界细节（空数组、越界、重复元素）。
- 第 3 轮：口述每个函数的循环不变量，再重新默写一次。

## 自检清单

- 二分查找是否正确处理空数组与单元素数组？
- mid 计算是否避免了整数溢出风险？
- 边界查找中，越界判断是否完整？
- 两数之和是否保证同一元素不被重复使用？
